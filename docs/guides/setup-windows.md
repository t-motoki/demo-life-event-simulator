# セットアップガイド（Windows）

## はじめに

このガイドは、**ターミナル（黒い画面）を使ったことがない方**を対象にしています。

完了後にできること：ライフイベント家計シミュレーターを起動し、Excel ファイルを出力できるようになります。

所要時間の目安：**約 30 分**

---

## 1. Python のインストール

Python はこのツールを動かすために必要なソフトウェアです。

### 方法A：公式サイトからインストール（推奨）

1. [https://www.python.org/downloads/](https://www.python.org/downloads/) を開く
2. 「Download Python 3.x.x」ボタンをクリックしてインストーラーをダウンロード
3. ダウンロードしたファイルをダブルクリックして起動
4. **「Add Python to PATH」のチェックボックスを必ずオンにする**（これを忘れると後の手順が動きません）
5. 「Install Now」をクリック

### 方法B：winget コマンドでインストール

すでにコマンドプロンプトを使えるかたは以下でもインストールできます。

```
winget install Python.Python.3
```

### インストール確認

コマンドプロンプト（次のセクションで説明）を開いて以下を実行し、バージョンが表示されれば成功です。

```
python --version
```

---

## 2. コマンドプロンプトの起動方法

「コマンドプロンプト」とは、キーボードで命令を入力してパソコンを操作する黒い画面です。このツールはここから起動します。

### 起動手順

1. キーボードの `Windows` キーを押す（または画面左下のスタートボタンをクリック）
2. 「cmd」と入力する
3. 「コマンドプロンプト」が表示されたらクリックして起動

---

## 3. ツールのダウンロード

### ZIPファイルでダウンロード（推奨・Gitなし）

1. GitHubのリポジトリページを開く
2. 緑色の「**Code**」ボタンをクリック
3. 「**Download ZIP**」をクリック
4. ダウンロードされた ZIP ファイルを右クリック →「すべて展開」→ 任意のフォルダに解凍

### git clone でダウンロード（Git インストール済みの場合）

Git をインストール済みの場合は以下のコマンドでもダウンロードできます。

```
git clone https://github.com/t-motoki/demo-life-event-simulator.git
```

---

## 4. 必要ライブラリのインストール

このツールは `openpyxl` というライブラリを使ってExcelファイルを生成します。

### 手順

1. コマンドプロンプトを開く（セクション2を参照）

2. ツールのフォルダに移動する。`cd` は「フォルダを移動する」命令です。

   例：デスクトップに解凍した場合
   ```
   cd C:\Users\あなたのユーザー名\Desktop\demo-life-event-simulator
   ```
   > **ヒント**：エクスプローラーでフォルダを開き、アドレスバーのパスをコピーして `cd ` の後に貼り付けると楽です。

3. 以下のコマンドを実行する
   ```
   pip install openpyxl
   ```

4. `Successfully installed openpyxl-...` と表示されれば完了です。

---

## 5. ツールの実行

1. コマンドプロンプトでツールのフォルダに移動（セクション4の手順2と同じ）

2. 以下のコマンドを実行する
   ```
   python src/main.py
   ```

3. 処理が完了すると、フォルダ内に Excel ファイルが生成されます。

---

## 6. Claude Code のインストール（オプション）

> **このセクションはツールを使うだけなら不要です。**
> Claude Code はこのツールを改修・カスタマイズしたい場合に使う AI エージェントです。

### 前提：Node.js のインストール

1. [https://nodejs.org/](https://nodejs.org/) を開く
2. 「LTS」版のインストーラーをダウンロードしてインストール

### Claude Code のインストール

コマンドプロンプトで以下を実行します。

```
npm install -g @anthropic-ai/claude-code
```

インストール後、APIキーの設定が必要です。詳細は [Claude Code 公式ドキュメント](https://docs.anthropic.com/ja/docs/claude-code/getting-started) を参照してください。

## 7. うまくいかないときは

### `python` が見つからない / `'python' は認識されていないコマンドです`

Python インストール時に「Add Python to PATH」にチェックを入れなかった可能性があります。
Python をアンインストールして、**チェックを入れて**再インストールしてください（セクション1参照）。

### `pip` が動かない

以下のコマンドを試してください。

```
python -m pip install openpyxl
```

### `cd` でフォルダに移動できない

フォルダのパスに日本語や全角スペースが含まれていると移動できないことがあります。ツールのフォルダを `C:\tools\` など英数字のみのパスに移動してから再試行してください。

### `No module named 'openpyxl'`

セクション4の手順でライブラリがインストールされていません。`pip install openpyxl` を再実行してください。

### それでも解決しない場合

GitHubの [Issues ページ](https://github.com/t-motoki/demo-life-event-simulator/issues) に状況を書いて報告してください。
