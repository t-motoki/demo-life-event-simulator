# Excel 出力フォーマットルール

作成日: 2026-04-07

---

## ファイル名

```
cf_simulation_YYYYMMDD.xlsx
```

実行日の日付を付ける。上書きしない。

---

## シート構成

| シート名 | 内容 |
| -------- | ---- |
| キャッシュフロー一覧 | 年次サマリー（メインシート・先頭に配置） |
| 入力内容確認 | 入力したパラメータの一覧 |

---

## キャッシュフロー一覧の列順（この順番で固定）

| 列 | 内容 |
| -- | ---- |
| 年 | 西暦 |
| 年齢（本人） | 本人年齢 |
| 年齢（配偶者） | 配偶者がいる場合のみ |
| 収入合計 | 本人給与 + 配偶者給与 + 年金 + その他 |
| 支出合計 | 生活費 + ローン返済 + イベント費用 + その他 |
| 住宅ローン控除 | 年末ローン残高 × 0.7%（13年まで） |
| 年間収支 | 収入合計 − 支出合計 + 住宅ローン控除 |
| 貯蓄残高 | 前年末残高 + 年間収支 |
| イベント | その年のライフイベント名（テキスト） |

---

## 表示形式

- 金額: 円単位・桁区切りあり（例: `1,234,567`）
- マイナス: 赤字・マイナス符号付き（例: `-500,000`）
- 貯蓄残高がマイナスの行: 背景色を薄い赤で強調
- 住宅ローン控除が 0 になる行（14年目以降）: 空白ではなく `0` を表示
