# 数値・金額フォーマットルール

作成日: 2026-04-07

---

## 金額

| 項目 | ルール | 例 |
| ---- | ------ | -- |
| 単位 | 円単位（小数点以下切り捨て） | `1,234,567` |
| 桁区切り | 3桁カンマ区切り | `12,345,678` |
| マイナス | マイナス符号 + 赤字表示 | `-500,000` |
| ゼロ | 省略せず `0` を表示 | `0` |

---

## 年齢・年数

| 項目 | ルール |
| ---- | ------ |
| 年齢 | 整数（小数点なし） |
| ローン年数 | 整数 |
| 控除適用年数 | 1〜13（13を超えたら0） |

---

## 計算精度

- 内部計算は **整数（円単位）** で行う
- 浮動小数点誤差を避けるため `round()` ではなく `int()` で切り捨て
- 年間収支の計算順: `収入合計 − 支出合計 + 住宅ローン控除`（この順を変えない）
