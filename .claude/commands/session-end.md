セッションを終了します。`tasks/current.md` を現在の作業状態で更新してください。

以下の手順で進めてください：

1. この会話でやったこと・今いる状態を把握する

2. `tasks/current.md` を以下の構成で上書き更新する：
   - **最終更新**: 今日の日付
   - **取り組み中のフェーズ**: フェーズ名とステータス（なければ「なし」）
   - **直前までやっていたこと**: 1〜3行で端的に
   - **ブロッカー・未決事項**: FP確認待ち事項を含む
   - **次回必要なファイル**: 最小限のリスト

   **「次にやること」は書かない。** 次のアクションは session-start 時に todo.md から毎回導出する。

3. このセッションで受けた指摘・修正があれば `tasks/lessons.md` を更新する：
   - 1〜2行で端的に（原因・NG行動・正しい対応）
   - 解消済みの教訓は削除してよい

4. このセッションで発見した残タスク・FP確認事項を `tasks/todo.md` に追記する

5. 変更があればコミットする（`/commit` を使う）

6. ユーザーへの報告を以下の形式で行う：

   **完了したこと**: （今回やったこと・1〜3行）
   **次回の起点**: （次のセッションで最初に取り組むべきこと）
   **持ち越し事項**: （FP確認待ち・未決事項があれば）
